📋 Get your free Firebase URL (2 mins):

  1. Go to console.firebase.google.com and sign in with Google
  2. Click Add project → name it ishanvi-savings → Continue → Create project
  3. In the left menu: Build → Realtime Database → Create Database
  4. Choose any location → select Start in test mode → Enable
  5. Copy the URL shown (e.g. https://ishanvi-savings-default-rtdb.firebaseio.com)
  6. Paste it below — do this once on each device!

Both you and Ishanvi paste the same URL once. After that it opens directly every time.
